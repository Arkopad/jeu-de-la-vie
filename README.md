## README

### Prérequis

Les librairies suivantes sont nécéssaires pour lancer le programme :

- tkinter

```sh
pip3 install tkinter
```

- numpy

```sh
pip3 install numpy
```

- tqdm

```sh
pip3 install tqdm
```

- numba

```sh
pip3 install numba
```

### Lancement 

1. Télécharger le dossier 
2. Executer le fichier 'MenuPrincpal.py'
